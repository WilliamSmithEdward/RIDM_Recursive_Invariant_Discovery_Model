# Recursive Invariant Discovery Model

RIDM is a restraint-governed, evidence-bound architecture for adaptive
reasoning and action.

## Specifications

- [RIDM 11.0](RIDM_11.MD) is the active specification. It adds evidence
  binding, explicit authority, dynamic materiality, bounded action and
  observation, and verifiable completion.
- [RIDM 10.0](RIDM.MD) is the historical restraint specification. It introduced
  minimum sufficient output and selective disclosure of material residuals.
